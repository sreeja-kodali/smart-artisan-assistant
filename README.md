# Smart-Artisan-Assistant
Smart Artisan Assistant is a web app that helps artisans manage daily production, materials, payments, and reports while providing AI-based price estimation, defect detection, and product quality guidance.

## Features
- Add daily production entries
- Track materials used
- Manage earnings and payment history
- Weekly and monthly reports
- AI-powered price estimation from product images
- Basic defect detection suggestions
- AI chatbot for product quality improvement tips

## Goal
To simplify daily tracking for artisans and provide basic AI-driven guidance for improving productivity and product quality.
